# kursi-atminas-spele

## Darbi

1. sastellēts git
2. uzlikts git gui GitKraken
3. paskatījos komandas terminālim

## Git komandas
Pievienot jaunas izmaiņas

* git add .
* git commit -m "komentārs kommitam"
* git push

Redzēt izmaiņas:

* git status

Iegūt izmaiņas no attalinātā repo (GitHUb)

* git pull
